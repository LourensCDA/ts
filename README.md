This is the project worked on in the Udemy course Understanding TypeScript - 2022 edition

Current Part 17

Setup:

1.  Install NodeJS
2.  Install TypeScript using npm

```ps
npm i -g typescript
```

3.  Run in folder

```ps
npm install
```

4.  Run lite server

```ps
npm start
```

To compile typescript

```ps
tsc app.js
```
